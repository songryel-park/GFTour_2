Good Feel Tour 보안 시스템

Spring Security 6.x 기반의 종합적인 보안 설정이 구현되었습니다.
사용자 인증, 권한 관리, 세션 보안, CSRF 보호 등 다양한 보안 기능을 제공합니다.

🔐 인증 & 권한

커스텀 로그인 페이지와 역할 기반 접근 제어를 통해 안전한 사용자 인증을 제공합니다.

🛡️ 세션 보안

세션 고정 공격 방지, 동시 세션 제어, 자동 만료 처리로 세션 보안을 강화했습니다.

🔒 CSRF 보호

모든 상태 변경 요청에 대한 CSRF 토큰 검증으로 크로스 사이트 요청 위조를 방지합니다.

📱 Remember-Me

2주간 유효한 자동 로그인 기능으로 사용자 편의성과 보안을 모두 확보했습니다.

🧪 보안 기능 테스트

아래 링크들을 클릭하여 설정된 보안 규칙들을 직접 확인해보세요:

관리자 페이지 (ADMIN 권한 필요) 관리자 API (ADMIN 권한 필요) 사용자 API (USER 권한 필요) 투어 목록 (공개 접근)
💡 테스트 가이드:
• 빨간 점: ADMIN 권한이 필요한 페이지 (관리자 계정으로 로그인 필요)
• 노란 점: USER 권한이 필요한 페이지 (로그인 필요)
• 초록 점: 누구나 접근 가능한 공개 페이지
